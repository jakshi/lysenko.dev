If you're wondering how much of your Claude Code budget is used:

```bash
brew install claudebar
```

Then run it and you'll have a nice monitor in your menubar.

Supports Claude, Codex, Gemini, GitHub Copilot, Antigravity, and Z.ai: https://github.com/tddworks/ClaudeBar
